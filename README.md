# Mintlify Website Clone (HTML & CSS)

This project is a **static clone of the Mintlify website**, built purely using **HTML and CSS**.  
The goal of this assignment was to replicate the **layout, structure, typography, and visual style** of the original Mintlify landing page as closely as possible, while strictly following the given constraints.

---

## Project Objectives

- Understand and replicate a modern documentation website layout
- Practice clean HTML structure and modular CSS styling
- Recreate real-world UI sections without using frameworks or JavaScript
- Focus on visual accuracy rather than functionality

---

## Constraints Followed

- HTML and CSS only  
- No JavaScript  
- No TailwindCSS or other CSS frameworks  
- No animations or fancy effects  
- No AI-generated content  
- Desktop-only layout (no responsiveness required)  
- Images and icons are chosen to closely resemble the original Mintlify site  
- Fonts, colors, spacing, and layout closely match the original design
